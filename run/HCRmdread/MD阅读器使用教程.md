# MD阅读器使用教程

## 1. 基本介绍

MD阅读器是HCR AAD模组提供的一个功能，用于在游戏内阅读Markdown格式的文档。通过这个功能，你可以查看游戏内的教程、注意事项和其他文档，无需离开游戏。
但注意，这只是一个简单的阅读器，作为游戏的教程阅读器使用，它不是一个专业阅读器，不具备专业阅读器的功能。
会在游戏开始时主动创建两个教程文件，但服务端的md文件并不会同步至客户端，请管理员注意

## 2. 如何打开MD阅读器

1. 进入游戏设置
2. 选择"按键控制"
3. 在"hcrpoints"分类下找到"打开MD文件阅读器"
4. 点击该选项，然后按下你想要设置的快捷键
5. 保存设置后，使用该快捷键即可打开MD阅读器

## 3. 使用方法

### 3.1 文件列表界面

当你打开MD阅读器时，首先会看到文件列表界面：
- 左侧显示所有可用的MD文件
- 使用鼠标滚轮上下滚动文件列表
- 点击任意文件名即可打开该文件
- 底部显示文件总数

### 3.2 文件内容界面

打开文件后，你会看到文件内容界面：
- **左侧为目录**：显示文件中的所有标题
  - 不同级别的标题有不同的缩进
  - 点击目录中的标题可以快速跳转到对应章节
  - 目录支持鼠标滚轮滚动
- **右侧为内容**：显示文件的具体内容
  - 使用鼠标滚轮上下滚动内容
  - 文本会自动换行，避免超出屏幕边界
  - 不同类型的文本显示不同颜色
- **顶部按钮**：
  - "返回列表"：返回文件列表界面
  - 中间显示当前文件名

## 4. 文本颜色说明

MD阅读器会根据文本内容自动添加不同的颜色：

### 4.1 标题颜色
- **一级标题**：红色（例如：# 主标题）
- **二级标题**：黄色（例如：## 子标题）
- **三级及以下标题**：绿色（例如：### 小标题）

### 4.2 内容颜色
- **警告信息**：橙色（包含"警告"、"Warning"、"WARN"等关键字）
- **错误信息**：红色（包含"错误"、"Error"、"ERR"、"ERROR"等关键字）
- **提示信息**：青色（包含"提示"、"Tip"、"注意"、"Note"等关键字）
- **列表项**：灰色（以"- "或"* "开头的行）
- **普通文本**：白色

## 5. 注意事项

1. **文件位置**：所有MD文件必须放在游戏目录下的"HCRmdread"文件夹中
2. **文件格式**：文件必须是UTF-8编码的Markdown格式文件（.md扩展名）
3. **标题格式**：使用#、##、###等符号表示不同级别的标题，这样才能在目录中正确显示
4. **中文支持**：完全支持中文显示，确保文件使用UTF-8编码
5. **快捷键设置**：默认没有快捷键，需要手动设置

## 6. 支持的Markdown语法

目前MD阅读器支持以下Markdown语法：
- 标题（#、##、###等）
- 普通文本
- 换行
- 列表（- 或 * 开头的行）

## 7. 常见问题解答

### Q: 为什么我看不到某些文件？
A: 请确保文件：
- 位于正确的"HCRmdread"文件夹中
- 使用.md扩展名
- 是UTF-8编码

### Q: 为什么目录中没有显示标题？
A: 请确保标题使用正确的Markdown格式，例如：
```
# 一级标题
## 二级标题
### 三级标题
```

### Q: 为什么中文显示乱码？
A: 请确保文件使用UTF-8编码保存

### Q: 为什么文本会自动换行？
A: MD阅读器会自动检测文本长度，当超过屏幕宽度时会自动换行，确保所有内容都能正常显示

## 8. 提示

- 你可以将游戏内的教程、攻略、注意事项等保存为MD文件，方便在游戏中查看
- 建议定期备份"HCRmdread"文件夹，避免文件丢失
- 可以使用外部Markdown编辑器编辑文件，然后复制到"HCRmdread"文件夹中使用
