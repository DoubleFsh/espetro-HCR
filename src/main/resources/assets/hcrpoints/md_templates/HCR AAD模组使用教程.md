# HCR AAD模组使用教程

## 1. 基本介绍

HCR AAD（Attack and Defense）是一个基于Minecraft Forge的模组，为游戏添加了团队攻防模式、战术地图、兵力系统等功能，适用于PVP服务器和团队游戏。

## 2. 核心功能

### 2.1 操作模式（Operation Mode）

操作模式是模组的核心功能，允许管理员启动一场攻防战：

#### 启动操作模式
```
/hcrpi start <总批次> [结束行为]
```
- `<总批次>`：行动的总批次数量
- `[结束行为]`：可选，当兵力耗尽时的行为
  - `terminate`：终止行动（默认）
  - `loop`：重启行动，开始新的循环

#### 结束操作模式
```
/hcrpi stop
```

#### 查看操作状态
```
/hcrpi status
```

### 2.2 战术地图

战术地图显示玩家位置、控制点和其他重要信息：

#### 打开/关闭战术地图
- 快捷键：默认V键（可在按键设置中修改）

#### 调整地图渲染范围
- 地图打开时按C键加大范围，按B键减小范围（50-1000格）

#### 打开地图配置界面
- 快捷键：默认X键（可在按键设置中修改）

#### 地图功能
- 显示玩家位置，不同队伍显示不同颜色
- 显示控制点位置和状态
- 支持缩放和旋转
- 可配置显示选项

### 2.3 兵力系统

每个队伍都有兵力值，兵力会随着时间或战斗情况变化：

#### 兵力规则
- 进攻方和防守方各有初始兵力
- 兵力会根据控制点的状态变化
- 当一方兵力耗尽时，行动结束

#### 低兵力警告
- 当一方兵力低于配置的阈值（默认10%）时，会播放警告音频
- 音频会在行动结束后5秒内逐渐淡出

### 2.4 团队系统

模组支持基于Minecraft队伍系统：

#### 队伍设置
- 进攻方：通常为"ATTACKER"队伍
- 防守方：通常为"DEFENDER"队伍
- 其他队伍的玩家会被警告并建议更换队伍

#### 玩家位置显示
- 战术地图上显示其他玩家的位置
- 友方玩家显示绿色
- 敌方玩家显示红色
- 可通过游戏规则控制是否显示其他玩家位置

## 3. 游戏规则

模组添加了以下游戏规则：

### 3.1 showPlayerLocation
- 控制是否在战术地图上显示其他玩家的位置
- 默认值：true
- 命令：`/gamerule showPlayerLocation true/false`

### 3.2 lowReinforcementThreshold
- 低兵力警告阈值，单位为百分比
- 默认值：10
- 可在配置文件中修改

## 4. 配置文件

模组使用Forge配置系统，分为客户端配置和服务端配置：

### 4.1 客户端配置
- 位于：`游戏目录/config/hcrpoints-client.toml`
- 包含：
  - 战术地图的显示选项
  - 音频设置
  - HUD显示选项

### 4.2 服务端配置
- 位于：`游戏目录/config/hcrpoints-common.toml`
- 包含：
  - 兵力系统设置
  - 操作模式默认值
  - 游戏规则默认值

## 5. 注意事项

### 5.1 管理员注意事项

1. **队伍设置**：确保在启动操作模式前设置好进攻方和防守方队伍
2. **兵力平衡**：根据玩家数量和地图大小调整兵力设置
3. **结束行为**：根据游戏需求选择合适的结束行为
4. **游戏规则**：根据服务器需求调整游戏规则
5. **配置备份**：定期备份配置文件，避免设置丢失

### 5.2 玩家注意事项

1. **队伍选择**：确保加入正确的队伍（进攻方或防守方）
2. **战术配合**：使用战术地图查看队友和敌人位置
3. **兵力意识**：关注队伍兵力情况，合理安排战术
4. **快捷键设置**：根据个人习惯设置合适的快捷键
5. **遵守规则**：遵守服务器管理员设置的规则和指令

## 6. 警告

1. **性能影响**：
   - 战术地图会占用一定的系统资源
   - 建议在低配置设备上适当降低地图显示选项

2. **游戏平衡**：
   - 不合理的兵力设置可能导致游戏失衡
   - 建议管理员根据实际情况调整兵力参数

3. **兼容性**：
   - 模组可能与其他PVP模组存在冲突
   - 建议在测试服务器上先进行兼容性测试

4. **数据安全**：
   - 模组数据存储在服务器上
   - 建议定期备份服务器数据

5. **滥用风险**：
   - 管理员指令具有较高权限
   - 建议只授予信任的玩家管理员权限

## 7. 常见问题解答

### Q: 为什么我看不到其他玩家的位置？
A: 请检查：
- 战术地图是否已打开
- showPlayerLocation游戏规则是否为true
- 你和其他玩家是否在同一维度

### Q: 为什么兵力没有变化？
A: 请检查：
- 操作模式是否已启动
- 控制点是否正常工作
- 队伍设置是否正确

### Q: 为什么音频没有播放？
A: 请检查：
- 游戏音量是否开启
- fightBGM文件夹是否存在
- 音频文件是否正确放置
- 低兵力阈值是否设置合理

### Q: 如何修改快捷键？
A: 进入游戏设置 -> 按键控制 -> 在"hcrpoints"分类下修改对应快捷键
